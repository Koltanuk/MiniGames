Mini - Games Platform

Choose one start having fun!

Bulls And Cows

Bulls and Cows is a classic code-breaking game. The objective is to guess a 4-digit secret number where each digit is unique.

Run & Jump

This game is a classic 'run and jump' style of game where you need to jump to overcome obstacles.

Memory Flip

Flip is a card memory game. Click the cards to see what color they uncover and try to find the matching color underneath the other cards. Uncover two matching colors at once to eliminate them from the game. Eliminate all cards as fast as you can to win the game.

MineSweeper

Minesweeper is a fun single player computer game. The goal of the game is clear the minefield by clicking safe squares and flagging the mines.. The rule of the game is simple, the number on a block shows the number of mines adjacent to it and you have to open all numbers.
